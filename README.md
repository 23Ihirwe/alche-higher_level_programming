# Python Programming Masterclass

Welcome to the **Python Programming Masterclass** repository. This project is a comprehensive, structured collection of Python scripts and modules designed to guide development from basic programming concepts to advanced object-oriented programming (OOP) principles.

Each directory focuses on a fundamental core concept of Python development, clean coding practices, and data structures.

##  Repository Structure

The project is organized into the following modules:

* **`python-hello_world`**: Complete beginner syntax, console printing, basic operations, and environment setup.
* **`python-if_else_loops_functions`**: Control flow structures, conditional logic, loops (`while`, `for`), and reusable function blocks.
* **`python-import_modules`**: External module integration, custom package imports, namespace management, and scoping.
* **`python-data_structures`**: Sequential data storage using lists, tuples, and fundamental index-based arrays.
* **`python-more_data_structures`**: Advanced data handling with sets, dictionaries, nested loops, and lambda operations.
* **`python-exceptions`**: Runtime error protection and edge-case handling using `try`, `except`, `else`, and `finally` blocks.
* **`python-everything_is_object`**: Deep dive into object mutation, data types, variable references, and memory allocation.
* **`python-classes`**: Object-Oriented Programming (OOP) foundation, data encapsulation, attributes, and custom methods.
* **`python-more_classes`**: Advanced class architecture, magic methods (`__str__`, `__repr__`), and static properties.
* **`python-inheritance`**: Code reusability implementation through subclassing, polymorphism, and `super()` mechanics.

##  Getting Started

### Prerequisites
A working Python 3 installation is required:
* Python 3.x 
* Any standard terminal, shell, or code editor (e.g., VS Code)

### Installation
Clone this repository locally:
```bash
git clone https://github.com
cd YOUR_REPOSITORY_NAME
```

---

##  Script Execution

1. Navigate to the targeted topic folder:
   ```bash
   cd python-hello_world
   ```
2. Run the specified file using the python interpreter:
   ```bash
   python3 script_name.py
   ```
